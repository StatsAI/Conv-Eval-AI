# Conv-Eval-AI
Streamlit powered REST API which uses ML + sLLM's to perform real-time intent classification, to determine what a user believes about themselves or a topic, over time. 

Interactive mode: real-time intent classification
JSON Mode: batch processing of user chat logs with an AI Agent 


Check out the app! https://conv-eval-ai.streamlit.app/
